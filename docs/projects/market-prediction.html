S&P 500 Market Prediction & Analysis

Tactical forecasting and defensible evaluation for S&P 500 returns

A project inspired by the Hull Tactical Market Prediction competition on Kaggle. It focuses on rigorous feature engineering, time-series-aware validation, and ensemble models to produce robust short-term signals for the S&P 500.

Kaggle Competition Source Code

Overview

Objective

Build predictive models and tactical signals for the S&P 500 that generalize across market regimes. Emphasis is placed on reproducible feature engineering, walk-forward validation, and risk-aware evaluation.

Data

Uses historical S&P 500 prices, derived technical indicators, macro features, and engineered statistics aligned to competition structure. Time-based features and careful alignment of targets prevent lookahead bias.

Feature Engineering

Rolling returns, volatility, momentum, ratio features, day-of-week encodings, and cross-asset signals where available. Features are normalized per-window to reduce regime sensitivity.

Validation

Walk-forward time-series split with out-of-time test sets. Backtests use transaction cost assumptions and position sizing rules to estimate realistic P&L.

Modeling

Gradient-boosted trees (LightGBM/XGBoost), stacked ensembles, and simple neural baselines where appropriate. Hyperparameter tuning via time-aware search.

Evaluation

Sharpe-like metrics, cumulative returns plots, precision/recall on directional predictions, and competition-specific scoring (as applicable).

Designed features to be robust across lookback windows and market regimes; use of winsorization and rolling normalization helped stabilize model inputs.

Blended multiple model families with time-aware cross-validation to reduce overfitting and improve generalization.
